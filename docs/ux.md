# UX

## Design Principles
- Prioritize glanceability over density.
- Keep critical flight metrics in consistent screen locations.
- Use icon + numeric pairings for rapid recognition.

## Information Priority
1. Battery / voltage status
2. Link quality (LQ)
3. RSSI / signal strength
4. Satellite count / GPS lock quality

## Interaction Model
- Default: passive telemetry display.
- Optional: profile-based view modes for race, cruise, and long-range.

## Visual Direction
- High-contrast foreground elements.
- Minimal decorative noise during flight.
- Predictable spacing and alignment across screen sizes.
